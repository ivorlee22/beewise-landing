# BeeWise Landing Page

Landing page tuyển người tham gia phỏng vấn trải nghiệm gia sư cho dự án BeeWise.

## Chạy dự án

```bash
npm install
npm run dev
```

## Kiểm tra

```bash
npm run lint
npm run build
```

## Deploy lên Vercel

Dự án sử dụng React và Vite. Trên Vercel:

- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

Không cần khai báo biến môi trường.
